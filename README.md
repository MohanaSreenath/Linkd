# Linkd

### To get this app you need to clone the repo first

1. git clone https://github.com/MohanaSreenath/Linkd.git
2. cd Linkd
3. chmod +x install
4. ./run

There you go!
You will the app to chat peer to peer chat fully encrypted via a secure and safe channel.
